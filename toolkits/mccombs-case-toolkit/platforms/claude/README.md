# McCombs Case Toolkit for Claude

**Version 1.3.0**

This plugin provides seven skills for the case-teaching lifecycle:

1. Case Idea Generator
2. Case Generator
3. Case Fact-Checker
4. Teaching Note Generator
5. Class Exercise Generator
6. Slide Generator
7. Case Refresher

Load this ZIP as a single Claude plugin. After installation, invoke the skills with natural prompts such as:

- “Brainstorm case ideas for my strategy course.”
- “Write a case about this company using the attached sources.”
- “Fact-check this case.”
- “Create a 75-minute teaching note.”
- “Design exercises for the case discussion.”
- “Build slides for this session.”
- “Refresh this case with current data.”

Each packaged skill is self-contained; shared McCombs references and generator dependencies are injected during the release build.
