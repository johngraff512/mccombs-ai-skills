# Install in ChatGPT

1. Build the toolkit or download the ChatGPT distribution bundle.
2. Unzip `mccombs-case-toolkit-chatgpt-v1.3.0.zip`.
3. Open each numbered folder and install its `skill.zip` separately.
4. Confirm that seven McCombs skills appear in the Skill library.

The outer distribution ZIP contains multiple skills and is not intended to be uploaded directly.

Suggested smoke tests:

- “Brainstorm case ideas for pricing strategy in an MBA course.”
- “Write a classroom case about this company using the attached sources.”
- “Fact-check this case before I create the teaching note.”
- “Create a 75-minute teaching note for this case.”
- “Design interactive exercises for this discussion.”
- “Create a slide deck for this case.”
- “Refresh this case with developments since 2024.”
