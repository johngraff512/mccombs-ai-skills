# McCombs Case Toolkit

A suite of seven AI-powered skills for case-based teaching at the McCombs School of Business, University of Texas at Austin.

## Overview

The McCombs Case Toolkit supports the full lifecycle of case-based teaching — from initial ideation through classroom delivery and case updates. Each skill produces professional, McCombs-branded outputs (Word documents and PowerPoint decks).

All skills work across McCombs departments: Strategy, Marketing, Finance, Operations, MIS, and Accounting.

## Skills

| Skill | What It Does |
|-------|-------------|
| **Case Idea Generator** | Brainstorm 3–5 case ideas for a course, topic, or framework |
| **Case Generator** | Write a full, classroom-ready business case document |
| **Case Fact-Checker** | Verify all factual claims in a case against primary sources before building downstream materials |
| **Teaching Note Generator** | Create an instructor teaching note with discussion plan, board plan, and timing |
| **Class Exercise Generator** | Design 5–8 interactive exercises (polls, debates, breakouts, etc.) for a session |
| **Slide Generator** | Build a McCombs-branded PowerPoint deck for case discussion or lecture |
| **Case Refresher** | Update an existing case with current data and recent developments |

## Workflow

The skills form a natural sequence, though each can be used independently:

1. **Case Idea Generator** — brainstorm which case to write
2. **Case Generator** — write the full case document
3. **Case Fact-Checker** — verify all claims against primary sources
4. **Teaching Note Generator** — create the instructor guide
5. **Class Exercise Generator** — design interactive activities
6. **Slide Generator** — build the slide deck for class
7. **Case Refresher** — update the case when data becomes outdated

## Trigger Phrases

Each skill responds to natural language. Examples:

- "Brainstorm case ideas for my strategy course"
- "Write a case about Tesla"
- "Fact-check this case"
- "Create a teaching note for this case"
- "What exercises should I use for this case discussion?"
- "Make slides for my case discussion"
- "Refresh this case with current data"
