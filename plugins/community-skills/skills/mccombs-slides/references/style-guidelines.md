# McCombs School of Business Slide Style Guidelines

These guidelines are derived from the official UT McCombs PowerPoint template and should be followed when creating presentations.

---

## Brand Colors

### Primary Palette
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| **Burnt Orange** | `#C05900` | Primary brand color, headlines, accent elements |
| **White** | `#FFFFFF` | Backgrounds, text on dark backgrounds |
| **Steel Blue/Gray** | `#9CACB6` | Secondary accent, subtle backgrounds |
| **Warm Gray/Tan** | `#D6D2C4` | Neutral backgrounds, subtle accents |

### Accent Colors
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| **Bright Orange** | `#F7961F` | Charts, highlights, callouts |
| **Yellow** | `#FFD600` | Charts, emphasis, warnings |
| **Green** | `#579C41` | Charts, positive indicators, sustainability |
| **Dark Blue** | `#005E86` | Links, charts, professional emphasis |
| **Teal** | `#00A8B6` | Followed links, secondary blue accent |

### Background Options
- **White** (`#FFFFFF`) — Standard content slides
- **Burnt Orange** (`#C05900`) — Title cards, section dividers
- **Charcoal/Dark** — Limited use for dramatic effect only

## Typography

### Font Families
- **Primary Font:** Arial (all weights)
- **Headline/Display Font:** Arial Black (for major headlines, all caps)
- **Secondary/Serif Font:** Georgia (for formal text or variety)

### Font Size Guidelines
| Element | Size | Weight | Case |
|---------|------|--------|------|
| Slide Title | 48pt | Bold (Arial Black) | ALL CAPS |
| Subtitle | 24pt | Regular | Sentence case |
| Body Text | 18pt | Regular | Sentence case |
| Supporting Text | 14-16pt | Regular | Sentence case |
| Footnotes/Captions | 12-14pt | Regular | Sentence case |

### Text Styling Rules
- Headlines on title cards: Arial Black, ALL CAPS, up to 3 lines maximum
- Body text: Arial Regular, sentence case
- Presenter names: Bold for name, regular for title/affiliation
- Avoid italics except for emphasis or titles of works
- Use bold sparingly for key terms only

---

## Slide Dimensions and Layout

### Slide Size
- **Widescreen 16:9** format
- Dimensions: 12,192,000 EMUs × 6,858,000 EMUs (approximately 13.33" × 7.5")

### Margins and Safe Zones
- Maintain consistent margins from slide edges
- Keep critical content away from the bottom where logos appear
- Left-align body text; center-align headlines on title slides

### Logo Placement
- McCombs logo appears in the lower-left corner on content slides
- Use white logo version on orange or dark backgrounds
- Use full-color logo on white backgrounds

## Slide Types and Layouts

### 1. Title Slide (Orange Background)
- Large headline in Arial Black, ALL CAPS, white text
- Subtitle in regular weight below a white dividing line
- Presenter name (bold) and title/affiliation
- Date in lower portion
- McCombs logo in burnt orange

### 2. Section Divider Slide
- Centered text on white or orange background
- Large statement headline
- Subtitle beneath explaining the concept
- Use for transitions between major sections

### 3. Content Slide (White Background)
- Title at top
- McCombs logo in lower-left
- Optional: Office or class name in header area

### 4. Two-Column Content
- Split layout for comparisons or dual information
- Equal column widths or 60/40 split for text+image

### 5. Chart/Data Slide
- Clean chart styling using brand colors
- Minimal gridlines
- Clear labels and legends
- Source citations in small text at bottom

### 6. Table Slide
- Simple table formatting
- Header row in brand colors
- Alternate row shading optional using light gray
- Avoid heavy borders

### 7. Image Slide
- Large high-quality imagery
- Text overlay on solid color band if needed
- Maintain aspect ratios

### 8. Infographic/Icon Slide
- Use simple, line-based icons
- Icons should be black or brand colors
- SVG format preferred for crispness
- Pair icons with brief explanatory text

## Content Guidelines

### Text Density
- **Maximum 1-3 points per slide**
- Slides complement the presentation; they are not for reading
- Use speaker notes for detailed content
- If you need more text, split across multiple slides

### Writing Style
- Use active voice
- Keep bullet points parallel in structure
- Front-load key information
- Avoid jargon unless audience-appropriate

### Visual Hierarchy
1. Headline captures the main idea
2. Subheadline provides context (if needed)
3. Body content supports with evidence or details
4. Source/footnote for attribution

## Charts and Data Visualization

### Color Usage in Charts
Apply brand colors in this order of preference:
1. Burnt Orange (`#C05900`)
2. Dark Blue (`#005E86`)
3. Steel Gray (`#9CACB6`)
4. Bright Orange (`#F7961F`)
5. Green (`#579C41`)
6. Yellow (`#FFD600`)

### Chart Styling
- Remove unnecessary gridlines
- Use data labels directly on charts when possible
- Avoid 3D effects
- Keep legends concise and positioned consistently
- Title every chart clearly

---

## Images and Icons

### Photography
- Use high-resolution images (minimum 150 DPI for print, 72 DPI for screen)
- Prefer authentic, diverse imagery
- Avoid clichéd stock photos (handshakes, pointing at screens)
- Campus and Austin imagery encouraged when relevant

### Icons
- Use simple, line-based icons with consistent stroke weight
- Black icons on light backgrounds
- White icons on dark/orange backgrounds
- SVG format for scalability
- Maintain consistent icon style throughout presentation

---

## Things to Avoid

- Clip art or low-quality images
- Walls of text
- Inconsistent fonts or colors outside the brand palette
- Heavy drop shadows or 3D effects
- Busy backgrounds that compete with content
- Overcrowded slides
- Fonts smaller than 14pt (except for source citations)

---

## Quick Reference Checklist

Before finalizing any presentation:

- [ ] Using Arial and/or Georgia fonts only
- [ ] Colors match McCombs brand palette
- [ ] McCombs logo is present and properly placed
- [ ] Images are high quality and appropriately sized
- [ ] Charts use brand colors consistently
- [ ] Text is readable (minimum 14pt)
- [ ] Consistent alignment and spacing throughout
- [ ] Spell-checked and proofread
